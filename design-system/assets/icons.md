# Aurora — Logo & icon notes

No logo exists in source. The mark at `logo.svg` is a minimal geometric placeholder using brand blue `#1C81EF`.

## Icons
- Source Figma references a Phosphor-like set by variant naming (`WeightRegular`, `WeightDuotone`, sizes 12/16/20/24).
- Use [Phosphor Icons](https://phosphoricons.com/) via CDN: `https://unpkg.com/@phosphor-icons/web@2.1.1`
- Default weight: `regular`. Alternate: `duotone` for promo/marketing.
- Size → weight mapping used in this kit:
  - Button XS (h28) — 16px icon
  - Button S  (h32) — 16px icon
  - Button M  (h40) — 20px icon
  - Button L  (h56) — 24px icon
