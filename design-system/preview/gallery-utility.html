Component gallery — vw-utility

Every class in vw-utilities.css — display, direction, alignment, gap, and grid helpers from core-mixin.scss. No new classes here, just how to combine the ones that exist.

1. vw-page-gap

.vw-page-gap sets gap: var(--vw-page-gap) (0.75rem / 12px) on a flex container — apply it on the container, not the children. Variants: .vw-page-gap-xl (20px), -md (12px), -sm (8px), -xxs (4px).

Block 1
Block 2
Block 3
<div style="display: flex;" class="vw-page-gap">
  <div>Block 1</div>
  <div>Block 2</div>
  <div>Block 3</div>
</div>

2. Cards in a row, spaced with vw-page-gap

8. Gap scale — vw-gap-*

.vw-gap-none / -xxs / -xs / -sm / -md / -lg / -xl — the general-purpose gap scale (distinct from .vw-page-gap-*, which is specifically for page-level flex spacing).

9. Grid column helpers

.vw-grid-cols-2 / -3 / -4 for fixed columns; .vw-grid-cols-auto-320 for a responsive auto-fill grid (the standard KPI-grid pattern).
