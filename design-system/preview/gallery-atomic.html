Component gallery — Layer 2 atomic

Form/table/menu primitives from components.css (the .nst-* family), plus embedded specimens for the atomic Figma-derived components that ship as standalone preview/*.html pages without a reusable class.

1. Buttons — .nst-btn

Base class = md size + outline variant. Add one size (--xs/--sm/--lg, default md) and one variant (--filled/--gray/--ghost/--action/--danger/--danger-subtle, default outline).

<button class="nst-btn nst-btn--filled">Filled</button>
<button class="nst-btn nst-btn--filled nst-btn--sm">Small</button>

9. Table — semantic <table> (.nst-table)

Prefer this for plain data grids — keeps screen-reader row/column semantics. Do not reuse .nst-table-row here (that's the grid variant's display:grid).

Acme dealActiveR. Iyer

  <thead><tr><th>Name</th>...</tr></thead>
  <tbody>
    <tr class="is-clickable"><td class="nst-table-td--primary">Acme deal</td>...</tr>
  </tbody>
</table>
